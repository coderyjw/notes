# 1. 软件的专业定义
- 软件的专业定义：一系列按照特定顺序组织的**计算机数据和指令**，是电脑的**非有型部分**
- 软件开发是什么呢？就是告诉计算机**一系列的指令**，这些指令也称之为**程序**。
- 开发软件的人就叫软件发开工程师。

# 2. 软件开发和应用程序开发的区别？
software application

软件包含应用程序 从下往上：操作系统 -> 编程语言 -> 算法 -> 应用程序

# 6. 网页的显示过程（基本过程 后面详细将）
1. 在浏览器输入url网站地址
2. DNS 服务器域名解析
3. 浏览器从服务器请求静态资源
4. 服务器返回给浏览器
5. 浏览器解析静态资源
6. 浏览器对静态资源进行解析和展示
