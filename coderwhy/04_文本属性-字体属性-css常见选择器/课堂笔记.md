# 1. CSS文本属性
1. text-decoration: 
   1. none 
   2. underline 
   3. overline 
   4. line-through
2. text-transform: 设置文字的大小写转换 
   1. capitalize 将每个单词的首字符变为大写
   2. uppercase 将每个单词的所有字符变为大写
   3. lowercase 将每个单词的所有字符变为小写
   4. none
3. text-indent: 缩进
4. text-align：定义行内内容（例如文本）如何相对它的块父元素对齐
   1. left
   2. center
   3. right
   4. justify 两端对齐
5. word/letter-spacing
