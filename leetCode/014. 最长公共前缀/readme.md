编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1：**

<pre><strong>输入：</strong>strs = ["flower","flow","flight"]
<strong>输出：</strong>"fl"
</pre>

**示例 2：**

<pre><strong>输入：</strong>strs = ["dog","racecar","car"]
<strong>输出：</strong>""
<strong>解释：</strong>输入不存在公共前缀。</pre>

**提示：**

* `1 <= strs.length <= 200`
* `0 <= strs[i].length <= 200`
* `strs[i]` 仅由小写英文字母组成
