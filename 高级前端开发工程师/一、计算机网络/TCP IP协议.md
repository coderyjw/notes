# 1. TCP/IP 协议群

- 应用层 Application 负责抽象用户数据，提供顶层的 API
- 传输层 Transport. host to host 点到点的传输
- 网络层 Internet 像物流公司一样 将封包从一个 ip 地址送到另一个 ip 地址
- 链接层 Link 设备到设备之间的数据传输
- 物理层 Pysical 光电信号的传输
