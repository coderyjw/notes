- 什么是react?
  用于构建用户界面的Javascript库（组件）（虚拟Dom）
- 为何选择React？
  - 组件化开发思想，项目便于维护
  - 只需关注业务逻辑，搞笑快速更新dom
  - 海量的周边生态，友好的开发环境
- React生态
  - React Redux
  - React Router
  - Dva
  - Ant Design
  - Style-component
  - React Native
  - TaroJs

- 学习路线
  - 基础（环境搭建、jsx、css、props）
  - 进阶（事件处理、state、生命周期、设计模式）
  - 生态工具（React-router Ant-Design）
  - 高级应用（Hooks、新生命周期、状态管理、ts应用）
  - 原理（虚拟dom、fiber、diff算法）
